### Getting Started

#### Prerequisites
* **JDK 8 or higher**: The `String.join()` method was introduced in Java 8.

#### Execution
1.  **Save** the file as `PrintOOPS.java`.
2.  **Compile** the code:
    ```bash
    javac PrintOOPS.java
    ```
3.  **Run** the program:
    ```bash
    java PrintOOPS
    ```
