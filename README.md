# OOPS-BannerApp

## Description
A simple usage of git bash and JDK, also basics of java

## Purpose
Creating a three diffrent branch for diffrent codes that uses to print "OOPS" in three ways
* Using system.out.println
* Printing OOPS in ASCII style
* Print OOPS in ASCII style using string.join

### Dependencies
* Text Editor (VSCodium/VSCode/NotePad++)
* JDK for javac,java cmds
* bash/zsh shell for CMD
* git bash

### Key CMDS
GIT bash
`git config user.name <your github Name>`
`git config user.email <your github email>`
`git init`
`git add`
`git commit -m "<commit message>"`
`git push -u origin <branchName>`
`git remote add origin <git.url>`
`git branch <branchname>`
